# Tokens

This is a table of tokens you have you can copy any token and use it in your application.

![](/images/tokens/page.png)


## Create a new token

To create a new token, click on the `New API token` button. You will be asked to provide a name .

![](/images/tokens/create.png)

One you provide the token name, click on the `Submit` button. you can show the token and you can copy it and use it in your application.

![](/images/tokens/token.png)


## Delete a token

To delete a token, click on the `Delete` button. You will be asked to confirm the deletion. Once you confirm, the token will be deleted and you will no longer be able to access it.

![](/images/tokens/delete.png)
